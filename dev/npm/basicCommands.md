# npm commands

- npm info jquery (check version history)
- npm view jquery (check registered info of package )
- npm un jquery (delete)
- npm update [-g] [...] 
- npm outdated (check outdated packages)

# 版本号前的 ^ ~  [major,minor,patch]

~ 

    会更新到minor version 中的最新版。

    eg: ~1.15.2     会更新到1.15.x 中的最新版本。 不会跟新到1.16.0

^ 
    会跟新到major version 中的最新版
    
    eg：^1.15.2     会更新到1.x.x中的最新版。 不会跟新到2.0.0
